INSTRUCTIONS D'INSTALLATION DE LA TODO APP:

1) EMPLACEMENT DES FICHIERS

Recopier l'intégralité des fichier et dossiers à l'interieur du dossier "proTask" 
vers un dossier situé dans le www de votre MAMP ou WAMP.

2) INTALLATION DE LA BASE DE DONNEES
Une fois le dossier créé, l'ouvrir et trouver le fichier "todo_app.sql".
Ouvrez ce fichier et copier l'intégralité de son code dans votre presse papier.
Dans votre navigateur de base de données,( phpMyAdmin ou autres),
vous connecter sur root et créer une base de données nomée "todo_app".
Collez ensuite tout le contenu de votre presse papier dans le serveur SQL de la base de 
données créée.
Ensuite, il faut créer un nouveau compte d'utilisateur de base de données, 
différent de root, et lui donner accès uniquement à la base "todo_app", 
évitant ainsi de mettre en péril les autres donées du compte root.
Modifiez alors le fichier "config_BDD.php" en accord avec vos informations de compte utilisateur de base de données.
